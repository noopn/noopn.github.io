#### 内部策略

支撑位不是一条线而是一个区间，在K线走出一个强势的图形后，如果在区间波动，可以标记该区间，如果K线突破了该区间，可以把这个区间看作支撑区间。

找到区间内的一个放量的K线，作为基础K线，在这个K线周围画出区间。

在区间内部买入，将止损位设置在区间上方或下方，止盈位置设置在相反的压力位。

#### 分型支撑区间

另一个找支撑区间的方法是，一段连续密集的K线算作一个分型，从分型中上影线最高位置到分型中上影线最低位置画支撑区间，或者是从分型中下影线最低位置到分型中下影线最高位置画支撑区间。

#### RSI

RSI (Relative Strength Index，相对强弱指数) RSI 的取值范围在 0 到 100 之间。

超买 (Overbought) > 70： 市场太热了，买方力量可能已经透支。就像油门踩到底太久，引擎快过热了，随时可能回调。

中轴线 = 50： 多空平衡点。

超卖 (Oversold) < 30： 市场太冷了，卖方卖得太狠。就像弹簧被压到了底，随时可能反弹。

TradingView 添加 RSI 后关闭中间边界，将上边界设置为 52 下边界设置为 48, 当 RSI 曲线突破这条线（进入 52 - 100 区间）时，市场进入了所谓的**超买区 (Overbought)**。

它不是做空信号： 看到 RSI 破了 52 就急着做空，这在强牛市里是自杀。价格可以带着 80 甚至 90 的 RSI 一路狂飙（这叫钝化）。要看的是它什么时候从上面掉回来。如果发现 ETH 处于疯牛行情，70 这个位置可能太敏感了。你可以去设置里把 RSI Upper Band 改成 80。只有突破 80，才算是真正的过热。

同时为 RSI 添加 EMA 指数加权移动平均线设置为 45， 并开启 RSI 的 SMA 设置为 9

加权逻辑： EMA 会给越靠近现在的价格分配越高的权重。

反应速度： 相比于普通的 SMA（简单移动平均线），EMA 45 对价格变动的反应更灵敏。如果 ETH 突然拉升，EMA 45 会比 SMA 45 更快地拐头向上。

使用这些指标的目的是， RSI 的移动平均线要比价格移动平均线快很多。 当 RSI 的 SMA9 上穿 EMA45 的时候可能是一个做多的机会，相反如果下穿可能是一个做空的机会。

#### Stoch

Gemini said
Stoch 是 Stochastic Oscillator（随机指标 / KDJ 指标的原型） 的缩写。

如果说之前 RSI 是一个衡量速度的体温计，那么 Stoch 就是一个衡量位置的百分比表。它的核心逻辑是：在上涨趋势中，价格倾向于收在近期波段的高点附近；在下跌趋势中，价格倾向于收在低点附近。

在随机指标在 60% 以上的时候，寻找做多的机会，在 40% 以下的时候寻找做空的机会，随机指标并不像 RSI 它可能连续几十天在 60% 上方， 或 40% 下方。

同时配合 200 移动平均线，当价格在 200 移动平均线以下的时候就不存在超卖，因此不能用 stoch 指标下穿 40% 来判断趋势要发生下跌， 但是可以用 stoch 在 60% 上方时来判断超买，因为价格在 200 移动平均线下并处于下跌趋势中，因此 stoch 在 60% 上方说明在当前的趋势中是超买的。

#### MACD

MACD (Moving Average Convergence Divergence，指数平滑异同移动平均线) 

DIF 线 (快线)： 通常是 EMA 12 减去 EMA 26 的差值。它反映了短期价格动能相对于长期的偏离程度。
DEA 线 (信号线/慢线)： 是 DIF 线的 9 日平滑平均值。它的存在是为了让信号更稳定，防止你被瞬间的插针干扰。

当柱子变长时： 说明快线正加速甩开慢线，趋势动能在增强。 当柱子变短时： 说明快线开始向慢线靠拢，趋势动能在减弱（也就是所谓的“动量衰减”）。

因为它都是基于 EMA（平均数）计算的，所以它永远比价格慢半拍。

在 0 轴下面是卖方的强势区，上面是买方的强势区，所以在买方强势区要做的是看涨，当 MACD 线上穿　signal 时是一个看涨的信号
相反在卖方强势区， MACD 线下穿　signal 时是一个看跌的信号

止损设置点的选择，因为 MACD 相对于价格通常是落后的， 所以买入点前面一定有更低的起始点，那这个点可以设置为止损位置。

通过 MACD 背离也可以判断趋势，价格有了更高的高点，但是 MACD 的有了一个更低的高点（柱状图或 MACD 线），虽然线是在 0 轴上方的买方强势区，但是趋势可能要反转。

相反价格有了更低的低点但是，MACD 有了更高的地点，虽然线是在 0 轴下方的卖方强势区，但是趋势可能要反转。

注意需要根据市场趋势进行，判断确保实在趋势进行中，如果一个 5 波浪的下跌趋势，最后已经不能创建新的地点，那么可以就不会继续下跌。

柱状图可以看动量的强弱，当价格到达阻力位置时，如果柱状图比较短，那么表示动量更弱，可能会突破。

#### 交易铁律

- 拥有经统计验证的基于规则的交易策略。
- 绝不要补仓
- 不要相信社交媒体的信息
- 必须设置止损点
- 必须设置无效点
- 只关注反转点形成的区域
- 只交易价格强势离开的区域
- 只交易价格曲线长度大于两倍时间长度的区域
- 只交易初次碰撞到阻碍的区域
- 🎯精准与稳定：专注于掌握单一交易策略，对其细节有深刻的理解。这种精准性使你能够自信且稳定地执行交易，从而提高成功率。
- 💡深入理解：深入研究一项策略，能让你全面了解其优势、劣势和细微差别。这种深入的理解能让你有效地适应各种市场状况。
- 📈适应能力：坚持一种策略，你就能熟练地判断何时最适合实施该策略，何时应该谨慎观望。这种适应能力在瞬息万变的加密货币市场中至关重要。
- 🧠心理自律：精通一种交易策略有助于培养交易纪律。你将不易受冲动决策的影响，更倾向于坚持你制定的计划。
- 📊业绩记录：专注的方法能让你积累运用特定策略的业绩记录。这种数据驱动的方法有助于你随着时间的推移不断改进策略。
- 🚀长期成功：成功的交易者通常将他们的成就归功于对单一策略的精通。这是长期维持盈利和发展加密货币交易投资组合的关键。

#### 框架 

**流动性** 过去的高低点，关键价位，SMC, 趋势线。

**巨量变化**， 价格总是朝着主要的流动性区域变化。

当价格触碰流动性区域，大量隐藏订单被激活，市场才能完成高效率的换手。机构投资者或“巨鲸”在进场或出场时，需要极大的成交量。如果他们在流动性枯竭的区域下单，会产生巨大的滑点（Slippage）。

锚定效应（Anchoring）： 交易者倾向于记住之前的历史最高点或最低点，并将其作为“安全”或“反转”的参考。

损失厌恶（Loss Aversion）： 绝大多数散户会将止损设置在“看起来很安全”的地方——通常是形态的边缘。这种集体的心理防线，在主力眼中就是一张清晰的“藏宝图”。

恐惧与贪婪的博弈： 当价格逼近这些区域时，心理压力会迫使散户交出筹码（投降），而这恰恰为市场提供了流动性。

算法掠夺： 量化机器人被编程为专门寻找这些止损池（Liquidity Pools）。一旦价格接近，算法会加速推升或打压，触发止损链条，从而在极短时间内完成头寸建立。

羊群效应（Herd Behavior）： 当价格朝着流动性区域移动时，技术分析派（看图说话的人）会看到同样的信号，从而产生协同行为。这种集体行为让预言自我实现。

#### 市场结构

突破：价格突破前高，或价格跌破前低，**趋势大概率会延续**。
结构转变：上涨趋势跌破前期低点，或下跌趋势突破前期高点，**通常视为结构反转的信号**。这种反转不一定直接反映在更大的趋势中，而是需要更近一步确认，例如 上涨趋势跌破前期低点后的反弹，并没有突破前面高点，那这种上涨趋势的反转概率会进一步增大。



#### 流动性

可以理解为限价单的数量，最终会成为别人的筹码。

从机构角度，当大单吃货的时候价格会必然上涨，因此他们会向下吃掉那些多头的止损单，用于平衡成本。因此会出现不断上移的低点的过程中，突然用阴线反包所有上涨趋势，吃掉哪些止损单。他们可能用各种原因来完成这一点，例如CPI数据，地区局势，美联储讲话。

价格总是从一个流动性区域，走向另一个流动性区域。 

想要确认流动性扫荡，需要看见突破后迅速回到内部流动性区间，然后在内部区间发生大的反向价格位移，这可能就是一次流动性扫荡，而不是突破。

#### 区间处理

划定时间框架 => 
找出框架内的**最近的**最高点和最低点，每一次向上或向下突破，**都形成新的价格区间** => 
画出水平线，找到中心点，也就是均衡线，上方为溢价区，下方为折价区 => 
在下跌趋势中，一般溢价区买入，卖在折价区，在上涨趋势中，一般折价区买入，卖在溢价区 => 
在高低点外就是流动性区域，价格总是在流动性所在的地方，来回切换

**两次高点位置相近，可以看作是一次流动性清扫，可能清扫后，价格迅速回落，因此不用看作是一个新的价格区间。**

**不同时间框架内的状态一致是概率最大的，比如都处于折价状态，且是上升趋势。如果日线是上涨的，但是小时线是下跌的就不要做空，而是等到他们状态一致**

**通常在公允价格区间，内部流动性区间寻找入场点，在外部流动性区间寻找止盈点，止盈点不应该在内部流动性区域。**
