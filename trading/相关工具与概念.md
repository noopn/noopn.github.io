#### 清算热力图 (Liquidation Heatmap)

[coinank](https://coinank.com/chart/derivatives/liq-heat-map) [coinglass](https://www.coinglass.com/pro/futures/LiquidationHeatMap?coin=ETH) 

表示的是大量挂单（Limit Orders）集中的价格区域。

清算水平 (Liquidation Levels)： 这是高倍杠杆交易者最怕的地方。如果你在 2200 刀开了 50 倍多单，你的清算价可能在 2160 刀。热力图会显示这些潜在的清算点位，因为一旦价格触及这里，会引发强制平仓（卖出），从而产生巨大的波动。

📌超短线交易：在价格接近高风险区域时进行短线交易⚡
📌突破与反转：当价格到达这些区域时，预计会出现剧烈的价格波动
📊 📌更合理的止损和止盈：明智地设置止损，避免过早被强制平仓⛔

📊 使用热图的真实交易案例

示例 1：价格进入高流动性区域
一位交易员注意到热力图上62,000 美元处出现了一大片黄色区域。当时价格在61,500 美元附近盘整，由于上方堆积了大量强制平仓订单，市场价格飙升至62,000 美元，引发强制平仓后回落。这种模式被称为流动性抢购，可用于预测价格走势。


示例 2：使用热图设置止损位
想要做空BTC/USDT的交易者会将止损位设置在主要清算区域上方略微偏上的位置，而不是区域内。这样可以防止止损位在预期反转发生之前被过早触发。热力图可以帮助交易者避开常见的清算陷阱。


例3：利用热图识别逆转
一位交易员发现价格在59,000 美元附近出现大量平仓，于是等待价格触及该水平。一旦市场暴跌并出现平仓，他们便注意到买盘压力正在增加。利用这一信号，他们在市场反弹上涨的瞬间建立多头头寸。


🎯 结语

清算热力图通过显示市场可能的走势，为交易者提供优势。使用清算热力图可以帮助您更好地预测趋势、管理风险并改进交易。✅

💡专业提示：结合清算热图、RSI指标、移动平均线和成交量分析，可以获得更精准的预测！📈

####　比特币市占率

BTC Dominance（简称 BTC.D，中文常译为“比特币市占率”或“比特币主导地位”）是加密货币市场中最重要的风向标之一。

$$BTC.D = \frac{\text{比特币市值}}{\text{全球加密货币总市值}} \times 100\%$$

| BTC 价格 | BTC.D (市占率) | 市场解读 (Money Flow) | 交易员建议 (Action) |
| :--- | :--- | :--- | :--- |
| **上涨** 🚀 | **上涨** 📈 | **BTC 独秀：** 资金高度集中在 BTC，正在吸干山寨币流动性。 | **只拿 BTC。** 严禁做多山寨币，防止“赚了指数赔了钱”。 |
| **上涨** 🚀 | **下跌** 📉 | **全面狂欢：** 踏入“山寨季”。资金外溢，小票涨幅远超大盘。 | **重仓 ETH/强势山寨。** 这是杠杆博取高额收益的最佳时机。 |
| **上涨** 🚀 | **横盘** ➡️ | **同步增长：** 市场全线上涨，BTC 与山寨币比例保持稳定。 | **均衡配置。** 持仓不动，享受整体市值增长。 |
| **下跌** 📉 | **上涨** 📈 | **避险模式：** 市场恐慌，资金从山寨币逃回 BTC 或兑换为 U。 | **空仓或只留 BTC。** 山寨币此时往往伴随断崖式阴跌。 |
| **下跌** 📉 | **下跌** 📉 | **大撤退：** 资金正从加密市场全面流向稳定币或离场。 | **空仓保命。** 泥沙俱下，不要试图去接下坠的飞刀。 |
| **下跌** 📉 | **横盘** ➡️ | **全线阴跌：** 整个板块同步走弱，没有明显的结构性机会。 | **减少操作。** 等待底部放量信号，保护本金。 |
| **横盘** ➡️ | **上涨** 📈 | **吸血洗盘：** 存量博弈，资金从山寨币抽回 BTC 避险。 | **卖出山寨。** 即使 BTC 不跌，山寨币也会因为流动性枯竭而走弱。 |
| **横盘** ➡️ | **下跌** 📉 | **结构性机会：** BTC 稳住，资金寻找低位山寨，补涨开始。 | **埋伏低位山寨。** 重点关注板块龙头、L2 或强庄标的。 |
| **横盘** ➡️ | **横盘** ➡️ | **垃圾时间：** 市场极度缩量，多空处于脆弱的平衡态。 | **观望为主。** 频繁短线会被手续费和波幅磨损殆尽。 |


#### 识别市场情绪

- 溢价指数 (Premium Index)，TrandingView 添加 ETHUSDT_PREMIUM
   
  溢价高： 大家都在加杠杆追涨，合约比现货贵很多，通常预示着即将回调。

  负溢价： 大家都急着抛售合约，合约比现货还便宜，可能已经见底。

- 资金费率 (Funding Rate) 
  
  正费率 (Positive)	多头 付钱给 空头看涨 (Bullish)	大家都想买，合约比现货贵。

  负费率 (Negative) 空头 付钱给 多头看跌 (Bearish)  大家都想卖，合约比现货便宜。
