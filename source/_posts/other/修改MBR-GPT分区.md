添加新的磁盘，或在虚拟机中扩容磁盘，需要修改分区。

#### 查看分区现状

```bash
lsblk
# 或者
fdisk -l /dev/sda

# 先使用 sudo apt install parted 安装 parted
parted /dev/sda print
```

可以看到磁盘容量变大，但是分区没有变化
