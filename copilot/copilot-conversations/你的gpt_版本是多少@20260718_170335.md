**ai**: 私有化部署 Nextcloud（自建私有云盘）最推荐、最便捷的方式是使用 **Docker 和 Docker Compose**。这种方式不仅安装简单，而且方便后续的升级和维护。

以下是使用 Docker Compose 部署 Nextcloud 的完整指南：

### 1. 准备工作

在开始之前，请确保您的服务器满足以下条件：
- 一台安装了 Linux（如 Ubuntu/Debian/CentOS）的服务器。
- 已安装 **Docker** 和 **Docker Compose**。
- （可选但推荐）一个域名，用于配置 HTTPS 访问。

---

### 2. 创建部署目录与配置文件

在服务器上创建一个专门存放 Nextcloud 数据的目录：

```bash
mkdir -p /data/nextcloud
cd /data/nextcloud
```

在该目录下创建一个名为 `docker-compose.yml` 的文件：

```bash
nano docker-compose.yml
```

将以下内容复制到文件中（这里使用 MySQL/MariaDB 作为数据库，性能比默认的 SQLite 更好）：

```yaml
version: '3'

services:
  db:
    image: mariadb:10.6
    container_name: nextcloud-db
    restart: always
    command: --transaction-isolation=READ-COMMITTED --log-bin=binlog --binlog-format=ROW
    volumes:
      - ./db:/var/lib/mysql
    environment:
      - MYSQL_ROOT_PASSWORD=your_root_password  # 替换为你的数据库 root 密码
      - MYSQL_PASSWORD=your_nextcloud_password  # 替换为你的 nextcloud 数据库密码
      - MYSQL_DATABASE=nextcloud
      - MYSQL_USER=nextcloud

  app:
    image: nextcloud:latest
    container_name: nextcloud-app
    restart: always
    ports:
      - "8080:80"
    links:
      - db
    volumes:
      - ./html:/var/www/html
      - ./data:/var/www/html/data
    environment:
      - MYSQL_PASSWORD=your_nextcloud_password  # 必须与上面 db 中的密码一致
      - MYSQL_DATABASE=nextcloud
      - MYSQL_USER=nextcloud
      - MYSQL_HOST=db
```

> **提示**：请将 `your_root_password` 和 `your_nextcloud_password` 替换为您自己设置的安全密码。

---

### 3. 启动服务

在 `docker-compose.yml` 所在目录下执行以下命令启动容器：

```bash
docker-compose up -d
```

启动后，Nextcloud 将在后台运行，并监听服务器的 `8080` 端口。

---

### 4. 配置 Nginx 反向代理与 SSL 证书

为了安全地从外网访问，建议使用 Nginx 进行反向代理，并配置 SSL 证书。

#### Nginx 配置示例
在您的 Nginx 配置文件中添加以下内容：

```nginx
server {
    listen 80;
    server_name pan.yourdomain.com; # 替换为你的域名
    return 301 https://$host$request_uri;
}

server {
    listen 443 ssl;
    server_name pan.yourdomain.com; # 替换为你的域名

    ssl_certificate /etc/nginx/conf.d/_club.crt; # 证书路径
    ssl_certificate_key /etc/nginx/conf.d/_club.key; # 私钥路径

    # 安全优化
    client_max_body_size 10G; # 允许上传大文件
    fastcgi_buffers 64 4K;

    location / {
        proxy_pass http://127.0.0.1:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        
        # 支持 WebSockets
        proxy_redirect off;
        proxy_buffering off;
    }
}
```

> **安全提示**：关于如何为您的域名申请和自动续期 SSL 证书，可以参考您之前的笔记 [[nginx证书自动续期]]，使用 `acme.sh` 工具来实现证书的自动化管理。

---

### 5. 初始化 Nextcloud

- 在浏览器中访问 `https://pan.yourdomain.com`（或 `http://你的服务器IP:8080`）。
- 创建管理员账号和密码。
- 数据库配置会自动完成（因为我们在 `docker-compose.yml` 中已经配置了环境变量）。
- 点击“安装”，等待几分钟即可进入 Nextcloud 仪表盘。

---

### 6. 常用维护命令

- **停止服务**：
  ```bash
  docker-compose down
  ```
- **重启服务**：
  ```bash
  docker-compose restart
  ```
- **升级 Nextcloud**：
  ```bash
  docker-compose pull
  docker-compose up -d
  ```
